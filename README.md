# Amazon-sales-PowerBI-dashboard
Executive Amazon Sales Dashboard built in Power BI using advanced features like drill-through, bookmarks, hierarchies, filters, and interactive visuals. Dataset includes Amazon Fashion &amp; Amazon Sales report.

## Business Objective

--Analyze overall sales performance
--Track revenue trends over time
--Monitor product-level performance
--Analyze units sold by date and category
--Provide an executive-level summary view

## Tools & Technologies Used

--Power BI Desktop
--Data Modeling
--DAX (Data Analysis Expressions)
--Data Cleaning & Transformation
--Interactive Visualizations

## Dashboard Pages & Features

1.  Sales Overview Page

--- Sales by Date (Line Chart), Total Sales KPI Card, Total Units Sold, Sales by Category, Dynamic Date Hierarchy (Year → Quarter → Month → Day), Interactive Slicers

2.  Product Review Page

--- Units Sold by Date, Category Performance, Drill-through functionality, Bookmark navigation, Zoom slider for trend analysis

##  Advanced Power BI Features Implemented

 Data Modeling (Relationships between tables),  Date Hierarchy, Drill-through Navigation, Bookmarks for Page Navigation, Interactive Filters & Slicers, KPI Cards, Custom Measures using DAX,  Dynamic Visual Interactions
